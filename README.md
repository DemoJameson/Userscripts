# 豆瓣影视添加 Trakt 待看按钮 🎬

一个用于在豆瓣电影和剧集页面添加 [Trakt.tv](https://trakt.tv/) 待看按钮的 Tampermonkey (篡改猴) 用户脚本。

## ✨ 功能特点

- **智能匹配机制**：优先通过豆瓣页面的 IMDb ID 进行精准匹配；若无 IMDb ID，则自动提取影视名称和年份进行后备搜索。
- **直观的 UI 集成**：在豆瓣电影/剧集的信息栏 (`#info`) 底部自然地嵌入 Trakt 对应条目的直达链接。
- **一键同步待看列表**：直接在豆瓣页面查看当前影视是否已在 Trakt 的 Watchlist 中。
  - 显示 `+ 待看`：点击添加到 Trakt 待看列表。
  - 显示 `- 移除`：点击从 Trakt 待看列表中移除。
- **安全的本地授权**：使用 Trakt 官方的 OAuth API 进行安全授权，Token 数据仅保存在本地 Tampermonkey 脚本管理器中。特制的弹窗 PIN 码输入框，防止切换标签页导致授权流程中断。

## 📦 安装指引

1. 首先需要在你的浏览器中安装 **[Tampermonkey (篡改猴)](https://www.tampermonkey.net/)** 或其他兼容的用户脚本管理器。
2. **[点击这里一键安装脚本](https://raw.githubusercontent.com/DemoJameson/Userscripts/main/douban-trakt.user.js)**。
   *弹出 Tampermonkey 安装提示后，点击“安装”或“升级”即可。得益于 `@updateURL` 属性设置，之后脚本均可自动获得更新。*

## 🚀 使用说明

1. 成功安装脚本后，打开任意一个豆瓣电影或剧集页面（例如：`https://movie.douban.com/subject/1292052/`）。
2. 在海报右侧的影视信息栏底部，你会看到新增的 **Trakt:** 字段和按钮。
3. **初次授权**：
   - 如果是第一次使用，按钮会显示为 `连接`。
   - 点击该按钮，会弹出一个自定义悬浮窗，同时在新标签页中打开 Trakt.tv 的官方授权页面。
   - 在新标签页中同意授权并获取你的 **PIN 码**。
   - 回到豆瓣页面，将 PIN 码粘贴到悬浮窗的输入框中并提交。页面随后会自动刷新。
4. **日常使用**：
   - 授权成功后，每次打开豆瓣页面，脚本会自动向 Trakt 查询并对比该片目。
   - 如果未添加，点击 `+ 待看` (红色按钮) 即可同步添加到你的 Trakt 账号。
   - 如果已添加，点击 `- 移除` (灰色按钮) 即可快速将其移出待看列表。

## 🔐 隐私与安全

本脚本**完全在本地运行**结构。你授权产生的 `access_token` 将使用 Tampermonkey 的内置储存 API (被限制在此脚本沙盒中) 保存在你电脑浏览器的本地存储里。本脚本除豆瓣和 `api.trakt.tv` 的必要请求外，**绝不会**与任何第三方服务器进行任何数据通信。
